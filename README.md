# 어디갔지?
어디갔지?는 지역 기반의 온라인 분실물 센터입니다. <br>
사용자들은 분실물과 습득물을 서로 공유하여 주인 잃은 물건의 주인을 빠르게 찾도록 도와줍니다. <br>

<br>

### 주요 기능
- 분실물 신고 접수 기능
- 분실물, 습득물 전체 조회
- 앱 내 포인트(화폐) 기능 (닻별, 별조각)
- 새로운 분실물 등록 시, 지역 기반의 푸시 알림 기능

<br>

### 사용 기술
- Spring Boot (Kotlin)
- Spring Data JPA
- Spring Security
- MySQL
- QueryDSL
- FirebaseCloudMessage
- AWS EC2, RDS, S3
